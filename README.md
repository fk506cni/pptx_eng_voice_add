## How to use

* タグで英語文章を囲んだパワーポイントファイルを用意する。上記のでもファイルを参照ください。
* ノートブック記載の処理により英語音声をねじ込む
* あるいは上記のWebアプリで英語音声をねじ込む
* VBAアプリでアニメーションを設定する

## memo
すでに音声データなどが挿入されたpptxファイルはpython-pptxでの音声挿入操作が失敗することがあるようです。
